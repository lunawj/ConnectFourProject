# ConnectFourProject
This is a Connect4 game written in Java for GVSU CIS457.  

To run the game:  
 - Compile the server: `javac GameServerMulti.java`  
 - Compile the client: `javac GameClient.java`  
 - Run the server: `java GameServerMulti \<port#\>`  
 - Run two instances of the client: `java GameClient \<server IP\> \<port#\>`
